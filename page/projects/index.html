Projects

Every once in a while, I’ve an idea that is worth to create a package about. A list of such projects below.

indexSort

This package–currently hosted on GitHub–contains functions to retrieve index sorted data and visually explore .fcs date files from various cell sorters.

It has been adapted as a plugin for FlowJo and is available via the FlowJo Exchange.

clustRcheck

This package–currently hosted on GitHub–contains statistical hypothesis testing for discovering the presence of more than one mode in cluster distributions.

It has been adapted as a plugin for FlowJo and is available via the FlowJo Exchange.
